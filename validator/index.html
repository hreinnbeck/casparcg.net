CasparCG config file validator

This is a simple validator for CasparCG config files.

Drag your casparcg.config file into the box below or paste in the contents of your config file below and hit . The results will display below.

For some insight into the CasparCG config file and it's options, check out "The Casparcg.config file explained" (still a work in progress).

Results:

Validates XML, verifies that only known tags are used, checks if user is using example code, verifies values of most tags. Summary with the number of channels and configured outputs.
